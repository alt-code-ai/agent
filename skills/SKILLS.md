## Prompts used to create skills

### fiction-writing-poetry-advanced

> Create a new skill, fiction-writing-poetry-advanced that provides expert guidance on writing literature 
> grade poetry, refer to and synthesise guidance from Writing Poems by Wallace and Boisseau, 
> On Poetry by Glyn Maxwell and other well reviewed / regarded poets and reference material. 
> Use the skill-creator skill, and use the exa mcp to search for extra context.  
> Build additional skill context as required for stronger guidance.  Evaluate the skill and install 
> ready for use (adjacent with existing fiction-writing-poetry skill and ready for performance comparison).

### fiction-writing-prose-advanced

> Create a new skill, fiction-writing-prose-advanced that provides expert guidance on

> techniques from well regarded sources such Janet Burroway (Writing Fiction), Stephen King,
> Ursula Le Guin, Gertrude Stein and others.  Use the skill-creator skill, and use
> the exa mcp to search for extra context.  Build additional skill context as required
> for stronger guidance.  Evaluate the skill and install ready for use
> (adjacent with existing fiction-writing-prose skill and ready for performance comparison)
